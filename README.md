# LunchGroup API 

API server which randomly groups people for lunch.

### Need
- configs/dbConfig.js
```
<sample>
module.exports = {
    db: {
        mongodb_user: 'xxxx@',
        mongodb_url: 'localhost:27017',
        db_name: 'xxxxx'
    },
    jwt: {
        key: 'xxxx',
        secret: 'xxxx'
    }
}
```

### build
- npm install
